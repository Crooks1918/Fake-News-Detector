## Overview 

Fake news refers to false or misleading information presented as legitimate news. It is often designed to appear credible but is ultimately created to deceive, manipulate, or generate revenue.

There are two main categories:

- Misinformation ‚Äì False information shared without harmful intent.
- Disinformation ‚Äì False information intentionally created and spread to deceive or cause harm.

This project develops an automated system that uses Natural Language Processing (NLP) and Machine Learning (ML) to classify news articles as credible or fake.
